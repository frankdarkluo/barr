## Observation

- strict Phase C 在移除显式 `Unknown` 提示之后仍然有效，说明干预效果没有因为去掉答案 hint 而消失。
- 在完整 `k=2/k=3` run 中，`k=2` 始终略强于 `k=3`，但优势是温和的，不是压倒性的。
- 在扩展 timing run 中，`k=2 t-2` 和 `k=2 t+1` 并列最强；`t+2` 没有再带来提升。
- 把 `random` 收紧成 `matched_random` 之后，transition-aware timing 和 control 的差距缩小了，但 `k=2 t-1/t/t+1` 仍然略高于它。
- `end_of_reasoning` 在当前 `k=2` 主实验里弱于 transition-aware timing，也弱于 matched random，但当前只覆盖了 `n=124`，还不是满覆盖。
- 文献风格 baseline (`E / E-50 / Rand`) 在 BBQ ambiguous 上几乎持平，说明在那个设置下，位置变化对最终分数影响很小。
- hidden-state separation 的最直观图是 centroid-axis view：`k=2 transition` 比 `k=3 transition` 分得更开，而且两者都比各自的 matched control 更分离。

## Hypothesis

- **[高]** 现在已经有比较强的证据支持较弱 claim：`intervention works`。也就是说，reflection-style cue 可以在不显式提示 `Unknown` 的情况下，把很多原本会 biased 的轨迹拉回正确答案。
- **[中]** `k=2` 仍然是最好的 first intervention candidate，但优势是局部且温和的，不宜写成特别强的 claim。
- **[中]** timing 仍然有价值，但当前真正需要比较的不是 `timing vs nothing`，而是 `timing vs generic reflection`；这一层证据目前还不够强。
- **[中]** 最有用的 timing window 可能不是单个精确 token，而是围绕第二个 transition 的一个短带状区域（大致 `t-2` 到 `t+1`）。
- **[低/待验证]** 如果在 originally-correct 组上也看到“correct retention 高、bias 不上升、但 token/latency 变短”，那更像是 selective correction；如果 correct 组被明显打坏，则更像 generic disruption。
